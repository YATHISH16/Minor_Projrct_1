🧬 WhatsApp GroupDNA Analyzer

WhatsApp GroupDNA Analyzer is a powerful Python-based data analytics tool designed to decode the "genetic makeup" of your group chats. By parsing exported WhatsApp chat logs, this application extracts deep behavioral insights, communication patterns, and relationship dynamics, turning raw text into interactive visual intelligence.

✨ Key Features
Activity Mapping: Track peak messaging hours, daily trends, and historical spikes.
Member DNA Profiles: Identify the "Group Lifeline" (most active), the "Lurker," and the "Media King."
Sentiment & Tone Tracking: Analyze emotional health and changing dynamics over time.
Lexicon & Emoji Analytics: Uncover the most frequently used words, catchphrases, and emojis.
Network Graphing: Visualize who interacts with whom most frequently through mentions and replies.

🛠️ Tech StackLanguage: 
Python 3.xData 
Processing: Pandas, NumPy, Regex

🚀 Quick Start

Export your WhatsApp chat as a .txt file (without media).
Clone this repository.
Run python analyzer.py and upload your file.
Explore your interactive dashboard!
